功能说明: 打印个人标识



[@打印个人标识]
<开关标识1/@开关标识(1)> <$flag(1)> << 显示 0 或 1\
<开关标识100/@开关标识(100)> <$flag(100)> << 显示 0 或 1\

[@开关标识]
#if
check [<$scriptparam1>] 0
#act
set [<$scriptparam1>] 1
sendmsg 7 ---------------标识:<$scriptparam1>，当前状态：<$flag(<$scriptparam1>)>（状态应显示为1）
goto @打印个人标识
#elseact
set [<$scriptparam1>] 0
sendmsg 7 ---------------标识:<$scriptparam1>，当前状态：<$flag(<$scriptparam1>)>（状态应显示为0）
goto @打印个人标识
