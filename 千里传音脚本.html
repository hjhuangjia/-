功能：
    使用传音命令无需佩戴传音道具.

格式：在没有佩戴传音道具前提下，使用命令进行千里传音

示列：

;===================获取聊天框所有文字=========================
[@UserCmd22]
#IF
CHECKGAMEGOLD > 0
#ACT
MOV S1 <$PARAMS>
GAMEGOLD - 1
SENDMSG 0 <$USERNAME>:<$STR(S1)> 255 253 0
#ELSEACT
SENDMSG 7 使用千里传音需要支付1元宝


;=======================以下为原始，不能使用空格，如需空格支持请看最后面一条============================================
[@UserCmd2]
#IF
CHECKGAMEGOLD > 0
#ACT
MOV S1 <$PARAM(1)>
GAMEGOLD - 1
SENDMSG 0 <$USERNAME>:<$STR(S1)> 255 253 0
#ELSEACT
SENDMSG 7 使用千里传音需要支付1元宝


[@UserCmd3]
#IF
CHECKGAMEGOLD > 0
#ACT
MOV S1 <$PARAM(1)>
MOV S2 <$PARAM(2)>
MOV S3 <$PARAM(3)>
MOV S4 <$PARAM(4)>
MOV S5 <$PARAM(5)>
MOV S6 <$PARAM(6)>
GAMEGOLD - 2
SENDTOPCHATBOARDMSG 0 149 235 30 <$STR(S1)><$STR(S2)><$STR(S3)><$STR(S4)><$STR(S5)><$STR(S6)>
#ELSEACT
SENDMSG 7 使用传音筒需要支付2元宝
